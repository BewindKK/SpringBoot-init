# SpringBoot-init
基本CRUD项目模板

# 模板组件
- SpringBoot 2.7.9
- Spring MVC
- Spring AOP
- MySQL 驱动
- MyBatis Plus
- Apache Commons Lang3
- 全局统一返回
- 全局异常处理
- 控制台日志内容扩充（AOP）
- Swagger + Knife4j 接口文档

# 使用
1. 修改文件目录名称
把“SringBoot-init”修改为你的项目名称，如“example-project”
> 在文件夹中修改，然后利用idea打开

2. 全局搜索把“SpringBoot-start”修改为你的项目名称

3. MySQL配置文件
修改用户名、密码、数据库地址、端口号
